# RadioButtonDemo - Pet Selection Application

## Description
This is a Java Swing-based GUI application that demonstrates the use of radio buttons for user interaction. The application allows users to select different types of pets using five radio buttons (Bird, Cat, Dog, Rabbit, and Pig) and displays a corresponding illustration of the selected pet.

## Features
- **Five Radio Buttons**: Bird, Cat, Dog, Rabbit, and Pig
- **Pet Illustrations**: Each selection displays a unique hand-drawn pet
- **Message Dialog**: Shows a confirmation message when a pet is selected
- **Default Selection**: Pig is selected by default
- **Clean GUI Layout**: Left panel for radio buttons, right panel for pet illustration

## Requirements
- Java 8 or higher
- Swing library (included in Java)

## How to Compile
```bash
javac RadioButtonDemo.java
```

## How to Run
```bash
java RadioButtonDemo
```

## Application Layout
- **Left Panel**: Contains 5 radio buttons for pet selection
- **Right Panel**: Displays the illustration of the selected pet
- **Interactive**: Selecting a radio button triggers a message box and updates the pet illustration

## Pets Included
1. Bird - Yellow bird with wings and beak
2. Cat - Orange cat with ears, whiskers, and tail
3. Dog - Brown dog with floppy ears
4. Rabbit - Light gray rabbit with long ears
5. Pig - Pink pig with snout, ears, and curled tail
